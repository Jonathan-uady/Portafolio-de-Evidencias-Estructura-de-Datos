La actividad consistió en un programa que interactuara con información que el usuario introducirá a manera de datos de cada estudiante por medio de un sondeo del hash

La complejidad de ésta ADA fue el conseguir que el sondeo se introduzca de manera correcta pues a veces no hacía nada y no interactuaba con los datos y se quedaba como un dato fantasma

Se pudo entender como se puede usar el sondeo lineal y cuadrático dependiendo de lo que se necesite para cada situación
